Glossaire
---------

.. glossaire::
  
   PHP

     * PHP est le langage de script utilisé par ProjeQtOr.

   MySql

    * MySql est le gestionnaire de base de données utilisé par défaut par ProjeQtOr.  
    * ProjeQtOr peut aussi utiliser une base de données de type PostgreSql.
